Errores comunes y soluciones

Soluciones rápidas para problemas frecuentes en POS Nómina.

Error 401: No autorizado

Problema

Error 401: Unauthorized - Invalid token or session expired
Solución:
  1. Limpia el caché del navegador (Ctrl+Shift+Supr)
  2. Cierra sesión completamente
  3. Inicia sesión de nuevo con tu usuario y contraseña
  4. Si persiste, solicita restablecimiento de credenciales

Error 500: Error interno del servidor

Problema

Error 500: Internal Server Error
Solución:
  1. Espera 5 minutos y recarga la página (F5)
  2. Prueba en otro navegador o en modo incógnito
  3. Verifica tu conexión a internet
  4. Si el error persiste, contacta a soporte e incluye el código de error

No puedo calcular la nómina

Problema

El sistema no genera el cálculo de nómina o muestra un error al procesar.

Solución:
  1. Verifica que todos los empleados tengan datos completos (nombre, documento, salario)
  2. Comprueba que las reglas salariales estén configuradas correctamente
  3. Asegúrate de que el periodo de pago esté dentro del rango permitido
  4. Intenta con un empleado de prueba para aislar el problema
  5. Si persiste, exporta los logs y contacta a soporte

Los reportes no se generan

Problema

Al intentar exportar un reporte, el navegador no descarga el archivo.

Solución:
  1. Permite que el navegador descargue archivos desde este sitio
  2. Verifica que tu carpeta de descargas tenga espacio disponible
  3. Desactiva bloqueadores de anuncios (pueden bloquear descargas)
  4. Intenta con otro navegador (Chrome, Firefox, Edge)
  5. Asegúrate de que los datos del reporte no estén vacíos

Datos incorrectos en la ficha del empleado

Problema

Los datos del empleado muestran valores incorrectos o desactualizados.

Solución:
  1. Recarga la página (F5) para obtener datos frescos
  2. Edita los campos y guarda los cambios
  3. Si los cambios no se guardan, comprueba tus permisos de usuario
  4. Intenta en modo incógnito para descartar problemas de caché
  5. Si el error persiste, contacta a tu administrador

La página carga lentamente

Problema

POS Nómina tarda mucho tiempo en cargar o responde lentamente.

Solución:
  1. Verifica tu velocidad de internet (requiere al menos 2 Mbps)
  2. Cierra otras pestañas y aplicaciones que consuman ancho de banda
  3. Limpia el caché del navegador (Ctrl+Shift+Supr)
  4. Desactiva extensiones que puedan ralentizar el navegador
  5. Intenta en otro dispositivo para aislar el problema

¿No encuentras tu error?

Si tu problema no está en esta lista, por favor:

  1. Describe el error de forma detallada
  2. Incluye los pasos que realizaste antes de que ocurra
  3. Adjunta capturas de pantalla
  4. Envía un ticket de soporte
