¿Cómo actualizar?

Las actualizaciones se instalan automáticamente. Después de actualizar:

  1. Limpia el caché de tu navegador (Ctrl+Shift+Supr)
  2. Cierra la sesión actual
  3. Inicia sesión nuevamente
  4. Verifica que los cambios se hayan aplicado correctamente

Nota: No se pierden datos durante las actualizaciones. Todos tus registros de empleados y nóminas se preservan.
